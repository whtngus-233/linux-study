# linux-study

리눅스(Ubuntu) 환경에서 직접 서버를 구성하고,  
사용자/권한/네트워크 문제를 해결하며 학습한 내용을 기록한 저장소입니다.

단순 명령어 정리가 아니라 실제 서비스 동작과 트러블슈팅 경험을 중심으로 구성했습니다.

---

## 실습 환경 아키텍처

[ Windows 11 ]
      │
      ├─ SSH (Port 22 / PuTTY) ──▶ [ Ubuntu 24.04 (VMware) ]
      │                                    │
      ├─ HTTP (Port 80) ──────────▶ [ Apache2 ] ──▶ /var/www/html/index.html
      │                                    │
      └─ FTP (Port 21) ───────────▶ [ vsftpd  ] ──▶ /home/user1/

---

## 1. 학습 개요
- 환경: Ubuntu 24.04 (VMware), PuTTY(SSH)
- 범위: 사용자 관리, 권한, 웹/FTP 서버 구축
- 목표: 단순 명령어 암기가 아닌 실제 서버 운영 경험 확보

---

## 2. 트러블슈팅

### 🔹 403 Forbidden 발생
- 문제: 웹 페이지 접근 불가
- 원인: index.html 권한 600
- 해결: chmod 644 index.html
- 결과: 정상 출력

웹 서버는 모든 사용자가 접근해야 하므로 other 읽기 권한이 필요하다는 점을 확인했다.

---

### 🔹 FTP 업로드 실패
- 문제: Permission denied
- 원인: vsftpd write_enable 비활성화
- 해결: 설정 변경 후 재시작
- 결과: 업로드 성공

서비스 설정 문제와 파일 권한 문제를 구분하는 것이 중요하다는 점을 이해했다.

---

### 🔹 ping 통신 불가
- 문제: 응답 없음
- 원인: Windows 방화벽 ICMP 차단
- 해결: ICMP 허용 설정
- 결과: 통신 정상

네트워크 문제와 서버 설정 문제를 구분하는 기준을 익혔다.

---

## 3. 실습 결과

- Apache 웹 서버 구축 및 페이지 배포 완료
- FTP 서버를 통한 외부 파일 업로드 성공
- 권한 변경에 따른 서비스 동작 변화 확인

---

## 4. 핵심 개념 정리

- chmod 755 → 디렉터리 접근 가능
- chmod 644 → 웹 서비스 필수 권한
- 디렉터리는 x 권한이 없으면 접근 자체 불가
- 권한 문제는 서비스 장애로 직결됨
- 문제 발생 시 권한 / 설정 / 네트워크 기준으로 원인 분석

### 📝 vi 에디터 명령어 정리

#### 1️⃣ 모드 전환 및 파일 조작 (EX 모드)
| 명령 | 설명 | 비고 |
| :--- | :--- | :--- |
| `:w` / `:q` | 저장 / 종료 | `:wq`로 동시 수행 가능 |
| `:w >> [파일명]` | 현재 내용을 기존 파일 끝에 추가 (Append) | 로그 기록 시 유용 |
| `:f` | 현재 파일 정보 및 라인 수 확인 | 작업 위치 파악 |
| `:! [command]` | vi를 나가지 않고 쉘 명령어 실행 | 편집 중 파일 목록 확인 등 |
| `:r! [command]` | 명령어 실행 결과를 현재 커서 위치에 삽입 | 실습 결과 기록 시 유용 |

#### 2️⃣ 효율적인 커서 이동 (명령 모드)
| 구분 | 명령어 | 설명 |
| :--- | :--- | :--- |
| **기본 이동** | `h`, `j`, `k`, `l` | 왼쪽, 아래, 위, 오른쪽 (방향키 대체) |
| **단어 단위** | `w` / `b` | 다음 단어 처음 / 이전 단어 처음으로 이동 |
| **행/문서 이동** | `0` / `$` | 행의 처음 / 끝으로 이동 |
| | `gg` / `G` | 문서의 최상단 / 최하단으로 이동 |
| | `[숫자]G` | **해당 라인 번호로 즉시 이동** (에러 수정 시 필수) |
| **화면 기준** | `H` / `M` / `L` | 현재 화면의 맨 위 / 중간 / 아래로 이동 |

#### 3️⃣ 편집 및 복사/붙여넣기
| 명령 | 설명 | 활용 팁 |
| :--- | :--- | :--- |
| `i` / `a` / `o` | 현재 위치 / 다음 칸 / 아래 줄 삽입 | 다양한 진입점 활용 |
| `x` / `X` | 커서 위치 문자 삭제 / 앞 문자 삭제 | - |
| `dd` | 현재 행 삭제 (잘라내기) | `[숫자]dd`로 여러 줄 동시 삭제 가능 |
| `yy` | 현재 행 복사 (Yank) | `[숫자]yy`로 여러 줄 동시 복사 가능 |
| `p` / `P` | 현재 행 이후 / 이전에 붙여넣기 | - |
| `dw` | 단어 단위 삭제 | 설정 값만 지울 때 유용 |
| `u` / `Ctrl + r` | 실행 취소 (Undo) / 다시 실행 (Redo) | 실수 방지 |

#### 4️⃣ 검색 및 치환 (Search & Replace)
| 명령 | 설명 |
| :--- | :--- |
| `/패턴` | 아래 방향으로 특정 문자열 검색 (`n`: 다음, `N`: 이전) |
| `?패턴` | 위 방향으로 특정 문자열 검색 |
| `:%s/Old/New/g` | **문서 전체에서 Old를 New로 일괄 치환** |
| `:10,20s/A/B/g` | 10~20번 라인 범위 내에서만 치환 |

#### 5️⃣ 사용자 편의 설정 (환경 설정)
| 명령 | 설명 |
| :--- | :--- |
| `:set nu` / `nonu` | 라인 번호 표시 / 해제 |
| `:set ts=4` | Tab 사이즈를 4칸으로 설정 |
| `:nohl` | 검색 결과 하이라이트 강조 해제 |
